# Construction Manager

> **Production-Ready** Construction Worker Health & Productivity Management System

## ✅ Features

- **Worker Management**: Add, view, edit, and delete workers
- **Scoring System**: Automatic fitness scoring based on height, weight, age, experience, and rating
- **Health Routines**: AI-generated exercise and dietary recommendations based on BMI
- **Report History**: Track worker reports with timestamps
- **PDF Export**: Generate downloadable PDF reports
- **Admin Dashboard**: Full worker management interface
- **Secure Authentication**: Database-backed admin login
- **Data Persistence**: All data saved to Turso database
